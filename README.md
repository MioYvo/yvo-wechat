## 微信聊天机器人 
基于 [itchat](https://github.com/littlecodersh/ItChat) 和 [图灵机器人](http://www.tuling123.com)。

## 使用方法
* `git clone `本项目
* 安装Python依赖包，推荐使用虚拟环境 virtualenv或conda

    ```
    virtualenv yvo-wechat
    source yvo-wechat/bin/activate
    ```
    
    安装依赖包
    
    ```
    pip install requirements.txt
    ```
* 启动

    ```shell
    python tuling.py
    // 会出现二维码
    ```
    
    打开微信扫描终端出现的二维码
